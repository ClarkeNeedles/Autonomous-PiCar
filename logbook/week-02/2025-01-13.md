### Session 2: Hardware Discussion (3:30 PM - 4:30 PM)

**Description**:
Team discussion on whether to keep the Raspberry Pi 4 or upgrade, and which software stack to use.

**Discussion**:
* Option A: Switch to Raspberry Pi 5. This offers better CPU performance for vision, but requires a high-current (5A) power supply and might have GPIO/HAT compatibility issues with the older PiCar chassis.
* Option B: Keep Raspberry Pi 4. It has native compatibility with the Robot HAT and is guaranteed to work with the provided battery setup.
* Software Stack: Briefly debated using ROS2 versus standard Python scripts. ROS2 is more robust but has a steeper learning curve.

**Decision**: Sticking with Raspberry Pi 4 to avoid electrical incompatibility with the provided robot chassis. If compute becomes a strict bottleneck later, we will attempt to migrate to the Pi 5. Further research will be done to determine the software stack.
